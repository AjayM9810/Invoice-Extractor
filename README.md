# Invoice-Extractor
Invoice Data Extractor is a Streamlit‑based OCR tool that processes PDF and image invoices, extracts line items and totals, validates computed amounts, and stores uploads in per‑user libraries. It provides a clean UI with login/registration, audit status indicators, and downloadable extracted data for quick review and reconciliation.
for testing 10 sample invoices will be given where 5 images .png files and 5 pdf files.
